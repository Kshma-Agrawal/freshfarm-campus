# Fresh Farm Campus – Interactive Prototype

This repository includes interactive Figma prototypes for both **Consumer** and **Farmer** modules of the **Fresh Farm Campus** platform.

The prototypes were created to simulate the user experience, validate navigation flows, and refine the interface before full implementation.

## Consumer Module – Simulated User Flows

The consumer prototype demonstrates the following key interactions:

* User login / authentication
* Browse nearby farmers
* Browse available produce
* View product details
* Add items to cart
* Checkout and order placement
* Order confirmation
* Order tracking

---

## Farmer Module – Simulated User Flows

The farmer prototype demonstrates the following key interactions:

* Farmer login and onboarding
* Farmer dashboard overview
* Add produce and manage inventory
* Order management
* Delivery and pickup coordination
* Access to resources and counselling
* Profile and earnings overview

---

## Purpose of the Prototype

The prototype was developed to:

* Visualize the complete user journey for both roles
* Validate navigation and interaction flows
* Ensure usability and consistency before development
* Support frontend implementation and testing

---

## Note

The prototypes represent the **design and interaction layer** of the system.
Actual functionality and system behavior are demonstrated through the **implemented frontend prototype and application code**.
