# Consumer Module – Interactive Prototype

An interactive prototype of the **Consumer Module** for the *Fresh Farm Campus* platform has been created in Figma.

This prototype demonstrates the key user flows and interactions for consumers purchasing farm produce from nearby farmers.

## Simulated User Flows

The prototype includes the following interactions:

• User login and onboarding
• Browse nearby farmers
• Browse available produce
• View product details
• Add items to cart
• Checkout and order placement
• Order confirmation
• Order tracking
