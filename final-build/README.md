## Demo Features

### Consumer Module
- Splash / onboarding flow
- Location selection
- Product browsing
- Category filtering
- Farmer discovery
- Cart and checkout flow
- Order confirmation and order details

### Farmer Module
- Farmer profile creation
- Produce management UI
- Order visibility
- Shared order data visibility from consumer side
- Farmer support chatbot

### Admin / Institutional Module
- Dashboard demo
- Farmer listings
- Marketplace section (demo)
- Counselling section (demo)
- Demand insights section (demo)
- Reports section (demo)

> **Note:** Some admin-side modules currently contain frontend/demo data because live institutional datasets and APIs were not available during development.
